# Amazon Sales Analysis Dashboard

## Overview
This project presents an interactive Amazon Sales Analysis Dashboard developed using Power BI. The dashboard helps analyze sales performance, customer behavior, product trends, and business insights through data visualization.

## Objective
The objective of this project is to transform raw Amazon sales data into meaningful insights that can support business decision-making.

## Tool Used
- Power BI
- Data Cleaning & Transformation
- Data Visualization
- Business Intelligence

## Dashboard Features
- Overall Sales Performance Analysis
- Product Category Analysis
- Revenue Insights
- Customer Purchase Trends
- Sales Distribution Visualization
- Interactive Filters and Slicers

## Key Business Insights
- Identified top-performing products and categories.
- Analyzed sales trends across different segments.
- Evaluated customer purchasing behavior.
- Generated actionable insights through visual analytics.
